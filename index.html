Tools Of The Trade

Projects

  • Personal Blog

    My personal blog created using Django(Python) and Bootstrap. The designing of the blog included a rough sketch followed by a wireframe made using Photoshop. Certain features like post categories were implemented on the backend.

  • MSIT.IN

    Official website of Maharaja Surajmal Institute of Technology. I coded the entire frontend of the website using Bootstrap.

  • IEEE-MSIT

    IEEE-MSIT's new revamped website. I coded the entire frontend using Materialize CSS. This was my first project which I built using a wireframe.

  • Frogger Game

    A game made using Object Oriented Concepts of JavaScript. I used vanilla JavaScript to handle multiple objects simultaneously. The game features multiple enemies that the player must avoid and collect gems to increase its score.

  • Random Quote Machine

    A web page that generates random quotes with the help of Adruxnet API. The quotations can also be shared on twitter at the click of a button.

  • Top 5 Bot

    A telegram bot which generates various top 5 listings. Currently the bot offers news in different categories, songs and trending movies with their cast and plot.

  • A Tribute Page For F.R.I.E.N.D.S

    My first attempt at a responsive website. The project is a tribute to the popular TV series F.R.I.E.N.D.S which never fails to entertain me. The project is made of pure HTML5 and CSS3.

  • YTS.AG Scraper

    A python script to scrape the famous yts.ag page which is used extensively to download English movies. The script displays the list of recent movies from the home page along with their IMDB rating and year of release. Upon entering the movie, a torrent download link will be generated succeeded by the synopsis of the movie.
